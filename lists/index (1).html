Web Development Hints

CSS (Cascading Style Sheets) allows things like the color of text, the background, the size, shape and position of all the different parts of a page to be set.
HTML stands for HyperText Markup Language, and is the language of the web.
JavaScript is a scripting language that allows designers to create interactions on a web page.
ARIA is an acronym for Accessible Rich Internet Application, and refers to the idea that web apps are used by a range of people, with differing needs for assistive technology and layout.
